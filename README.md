mcp-dl-list
===========

Scrapes tweets for MCP download links.

Create a `config.json` file containing these lines:

    {
      "consumer_key": "...",
      "consumer_secret": "...",
      "access_token_key": "...",
      "access_token_secret": "...",
    }

See the python-twitter docs for more info.

Usage:

    python pulltweets.py
